# **App Name**: FolioFlow

## Core Features:

- Project Showcase: Display featured projects with descriptions and visuals.
- Skills Summary: A section that concisely lists key skills and technologies.
- About Me: A brief overview of the individual, including experience and background.
- Contact Information: A clear way for visitors to get in touch.
- AI-Powered Content Suggestion Tool: An AI tool that assists the user in crafting the 'About Me' section by suggesting relevant skills and experiences based on keywords or industry.
- Theme selector: The user can decide from prebuilt light and dark themes

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Very light blue (#E8EAF6), providing a soft, unobtrusive backdrop.
- Accent color: Violet (#7E57C2), creating a contrasting but harmonious accent for interactive elements.
- Body and headline font: 'Inter', sans-serif, known for its clean and readable design, suitable for both headlines and body text.
- Use minimalist icons for skills and contact information.
- Clean, single-page layout with clear sections and smooth scrolling.
- Subtle transitions and animations on hover effects to improve user experience.
